HTML Editor

A custom built html editor



HTML Usage:

You can instance an html-editor element directly from HTML by simply adding the attribute [data-modux-component="topsmith-component-html-editor"] to an element. For this to work, you need to load the html-editor component in the application: topsmith.registerComponent( ComponentHtmlEditor )



JS Usage:

Considering we have the following element:



In order to create the html-editor component on the element, you would need to add this to the execute() method of your page controller.
