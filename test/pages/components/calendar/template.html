Calendar

A custom built date picker



HTML Usage:

You can instance a calendar element directly from HTML by simply adding the attribute [data-modux-component="topsmith-component-calendar"] to an element. For this to work, you need to load the calendar component in the application: topsmith.registerComponent( ComponentCalendar )



JS Usage:

In order to create the calendar component on the element, you would need to add this to the execute() method of your page controller.



You can customize the component by changing the properties via HTML attributes or js with object.set( property, value)
