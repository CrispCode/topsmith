Clock

A custom built analog clock



HTML Usage:

You can instance a clock element directly from HTML by simply adding the attribute [data-modux-component="topsmith-component-clock"] to an element. For this to work, you need to load the clock component in the application: topsmith.registerComponent( ComponentClock )



JS Usage:

Considering we have the following element:



In order to create the clock component on the element, you would need to add this to the execute() method of your page controller.



You can customize the component by changing the properties via HTML attributes or js with object.set( property, value)
